FORTUNE BUSINESS SOLUTIONS - API CONFIGURATION GUIDE
====================================================

This document contains all the required API keys and configuration steps needed to run the Fortune Business Solutions website in production.

====================================================
REQUIRED ENVIRONMENT VARIABLES
====================================================

1. RESEND_API_KEY (Email Service)
   --------------------------------
   Purpose: Send contact form submissions to ceo.fortune@outlook.com
   
   Setup Instructions:
   a) Go to https://resend.com/
   b) Sign up for a free account (allows 100 emails/day, 3,000/month)
   c) Verify your domain OR use the testing domain provided by Resend
   d) Go to API Keys section in dashboard
   e) Create a new API key
   f) Copy the API key (starts with "re_")
   
   Add to Vercel:
   - Go to your Vercel project settings
   - Navigate to Environment Variables
   - Add: RESEND_API_KEY = re_xxxxxxxxxxxxx
   - Set for Production, Preview, and Development environments
   
   Notes:
   - The contact form will work in "simulation mode" without this key for testing
   - For production, you MUST add this key to receive actual emails
   - Emails are sent to: ceo.fortune@outlook.com
   
   Testing:
   - Test the contact form after deployment
   - Check spam folder if emails don't arrive
   - Verify sender domain is properly configured in Resend

2. Google Maps API (Currently using iframe embed)
   -----------------------------------------------
   Current Status: Using Google Maps iframe embed (free, no API key needed)
   Location: Ruwi, Muscat, Oman
   
   If you want advanced map features:
   a) Go to https://console.cloud.google.com/
   b) Create a new project
   c) Enable Google Maps JavaScript API
   d) Create API credentials
   e) Add: NEXT_PUBLIC_GOOGLE_MAPS_API_KEY = your_api_key
   
   Note: Current iframe implementation is sufficient for most needs


3. Instagram API (For Social Page)
   ----------------------------------
   Current Status: Links to @fortune_oman Instagram profile
   
   If you want to fetch live Instagram posts:
   a) Go to https://developers.facebook.com/
   b) Create a Facebook App
   c) Set up Instagram Basic Display API
   d) Add: INSTAGRAM_ACCESS_TOKEN = your_token
   
   Note: Manual updates recommended for curated content

====================================================
SUPPORT & TROUBLESHOOTING
====================================================

Contact Form Issues:
- Verify RESEND_API_KEY is set correctly
- Check Vercel function logs for errors
- Ensure email recipient is correct
- Test with simulation mode first

Map Not Loading:
- Check internet connection
- Verify iframe URL is correct
- Test in different browsers
- Clear browser cache

Video Not Playing:
- Verify fortune-logo.mp4 is in public folder
- Check video codec compatibility
- Test on different browsers
- Ensure video file is not corrupted

Navigation Issues:
- Clear browser cache
- Check scroll-to-top component is working
- Verify all route paths are correct
- Test on different devices
